Permet de publier automatiquement vos articles sur votre jblog Jcray.

== Description ==

<strong>AutoJblog</strong> publie automatiquement vos articles wordpress sur votre jblog Jcray (pour plus d'informations cliquez ici <a href="http://gregoire-penverne.fr/1824-jbloguez-automatiquement-a-partir-de-votre-wordpress">http://gregoire-penverne.fr/1824-jbloguez-automatiquement-a-partir-de-votre-wordpress</a>


== Installation ==

Extraire le fichier zip et déposez simplement le contenu dans le répertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins.
